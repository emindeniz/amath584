# THE FUNCTION TO READ THE IMAGES:
def read_pgm(pgmf):
    pgmf = open(pgmf, 'rb')
    """Return a raster of integers from a PGM as a list of lists."""
    assert pgmf.readline() == b'P5\n'
    (width, height) = [int(i) for i in pgmf.readline().split()]
    depth = int(pgmf.readline())
    assert depth <= 255

    raster = []
    for y in range(height):
        row = []
        for y in range(width):
            row.append(ord(pgmf.read(1)))
        raster.append(row)
    return raster

2. What is the interpretation of the U, Σ and V matrices? (Plot the first few reshaped columns of U)?¶

Asnwer:¶

In a typical SVD samples are in rows and features are in columns. Here we are instucted to put the image vectors as columns in to the data matrix which we means that they are our features. Each pixel in the data matrix has a different value for each image. When we perform SVD, we aim to reduce this representation such that each pixel can be represented by fewer number of summary images. So U is pixels to summary images matrix. Σ is the strength of each summary image and V is images to summary images matrix that tells us the contribution of each image to the each summary image. 4. compare the difference between the cropped (and aligned) versus uncropped images in terms of singular value decay and reconstruction capabilities.¶

Answer:¶

I see the following differences:¶

1) In terms of SVD spectrum, more components are needed to fully cover the variation in pixels. The reason could be that the images are less standardized. (For examples 200/1600=0.125 or 12.5% of the components cover 80% of the variation in cropped images vs. 50/165=0.30 or 30% of the components are needed to cover 80% of the variation in uncropped images).¶

2) Average images look very different. The average images in cropped images focus on details on each image like eyes, mouth etc. The average imagse in uncropped images focus on the fact that there is an head in each image. I think the difference is due to the fact that the most of the variation in cropped images is within the face itself and the everything else is standard. Compare that with uncropped images, there is quite a bit of variation in terms of the position of the head within the image.¶
